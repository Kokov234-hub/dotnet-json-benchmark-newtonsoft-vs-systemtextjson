# .NET JSON Benchmark: Newtonsoft.Json vs System.Text.Json

Количествен бенчмарк (Serialize/Deserialize) с BenchmarkDotNet в .NET 8.

## Стартиране
```bash
dotnet restore
dotnet run -c Release
```
## Версии/среда
- .NET SDK: 8.x
- BenchmarkDotNet: 0.15.5
- Newtonsoft.Json: 13.0.4
- OS/CPU/RAM: macOS <твоята версия>, <CPU>, <RAM>
- Run mode: Release, x64
